# Printer Queue Simulation

This project implements a **Printer Management System** using the **Queue** data structure. It demonstrates the First-In-First-Out (FIFO) principle, ensuring that print jobs are processed exactly in the order they are received.

## Features
* **Queue Operations:** Custom implementation of `enqueue`, `dequeue`, `peek`, and `isEmpty`.
* **Printer Logic:** A `PrinterQueue` wrapper class that manages job assignment and sequential processing.
* **FIFO Enforcement:** Ensures fair processing of documents based on arrival time.

## Getting Started

### Prerequisites
You need **Node.js** installed on your computer to run this code directly in your terminal, or you can run it in your browser's Developer Tools Console.

### Running the Code
1. Copy the code provided for the `Queue` and `PrinterQueue` classes.
2. Paste it into a file named `printer.js`.
3. Open your terminal or command prompt.
4. Run the code using Node.js:
   ```bash
   node printer.js
